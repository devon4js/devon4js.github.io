The Open Application Standard Platform provides a solution to building applications which combine best-in class-frameworks and libraries as well as industry proven practices and code conventions. devon4JS is the JavaScript variant of the devonfw focusing on JavaScript client's part of applications

Application Starters

devonfw offers you a template based on Angular 4, Material Design and Covalent. The application starters define a structure as well as integrates best-in-class frameworks and libraries. In addition to this, the template includes Angular CLI scripts which support developers in their activities such as coding, testing and building the application.

Getting started

Follow the getting started tutorial

The devon4JS getting started guide introduces you to devonfw and in particular devon4JS. You'll learn about a devon4JS application's overview, structure and components, based on our My Thai Star reference application. Afterwards you'll be guided through the process of building your first devon4JS application from scratch, Jump the Queue.

Use our component catalog

Our component catalog contains useful components from Material Design, Bootstrap, PrimeNG and more.

Use the devon4JS template to start a new project

If you're already familiar with all the technical details and just want to start developing a devon4JS application, use the devon4JS application template.

View the sample application

Take a look at the devon4JS client in action with our reference application and showcase "My Thai Star"
